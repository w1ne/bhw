# Budapest Hardware Club

The website for the Budapest Hardware Club: a bilingual (EN/HU) community for people building hardware, firmware, embedded systems and robotics.

## Local development

```bash
npm install
npm run dev
```

Run the production checks with:

```bash
npm test
npm run build
```

## Deployment

Cloudflare Pages deploys this static Astro site from GitHub.

- Production branch: `main`
- Build command: `npm run build`
- Build output directory: `dist`

Every push to `main` starts a deployment.
